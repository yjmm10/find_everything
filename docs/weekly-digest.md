# 📊 技术周报 [2026-04-01 ~ 2026-04-07]

## 📄 Arxiv 前沿论文

**1. In-Place Test-Time Training: Adapting LLM Weights Dynamically**
提出在部署阶段原地微调LLM权重，打破传统"训练-部署"静态范式，使模型能实时适应上下文变化。
🔗 http://arxiv.org/abs/2604.06169v1

**2. Exclusive Unlearning: Targeted Knowledge Removal in LLMs**
针对医疗、教育等高风险场景，提出选择性遗忘技术，有偿移除特定有害知识同时保留核心能力。
🔗 http://arxiv.org/abs/2604.06154v1

**3. LLM4CodeRE: Generative AI for Code Decompilation**
首个将LLM应用于恶意软件逆向工程中代码反编译分析的任务，突破传统静态分析局限。
🔗 http://arxiv.org/abs/2604.06095v1

**4. CritBench: LLM Cybersecurity Evaluation for IEC 61850**
构建电力系统数字变电站环境的LLM安全评估框架，填补关键基础设施AI安全基准空白。
🔗 http://arxiv.org/abs/2604.06019v1

**5. From Hallucination to Structure Snowballing: Constrained Decoding Alignment**
揭示约束解码中"幻觉雪球"现象，提出结构化滚动方法提升LLM反思能力。
🔗 http://arxiv.org/abs/2604.06066v1

**6. A Multi-Stage Validation Framework for Clinical Information Extraction**
构建可信临床信息提取的LLM验证管道，解决医疗记录非结构化数据到临床知识的转化难题。
🔗 http://arxiv.org/abs/2604.06028v1

**7. Context-Value-Action Architecture for Value-Driven LLM Agents**
提出价值驱动型LLM Agent架构，解决现有Agent行为僵化问题，增强目标导向决策能力。
🔗 http://arxiv.org/abs/2604.05939v1

**8. FrontierFinance: Long-Horizon Financial Tasks Benchmark**
首个评估LLM在真实金融任务中计算机使用能力的基准，填补AI劳动力替代研究空白。
🔗 http://arxiv.org/abs/2604.05912v1

**9. BOSCH: Black-Box Binary Optimization for Attention-Head Selection**
通过二进制优化方法自动选择短上下文注意力头，在保持性能前提下显著降低推理KV缓存开销。
🔗 http://arxiv.org/abs/2604.05942v1

**10. TriAttention: Trigonometric KV Compression for Long Reasoning**
针对长推理场景的KV缓存瓶颈，提出三角函数压缩方法，高效处理长上下文推理。
🔗 http://arxiv.org/abs/2604.04921v1
