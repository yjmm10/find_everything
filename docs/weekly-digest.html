3. LLM4CodeRE: Generative AI for Code Decompilation 首个将LLM应用于恶意软件逆向工程中代码反编译分析的任务，突破传统静态分析局限。 🔗 http://arxiv.org/abs/2604.06095v1

5. From Hallucination to Structure Snowballing: Constrained Decoding Alignment 揭示约束解码中"幻觉雪球"现象，提出结构化滚动方法提升LLM反思能力。 🔗 http://arxiv.org/abs/2604.06066v1

6. A Multi-Stage Validation Framework for Clinical Information Extraction 构建可信临床信息提取的LLM验证管道，解决医疗记录非结构化数据到临床知识的转化难题。 🔗 http://arxiv.org/abs/2604.06028v1

7. Context-Value-Action Architecture for Value-Driven LLM Agents 提出价值驱动型LLM Agent架构，解决现有Agent行为僵化问题，增强目标导向决策能力。 🔗 http://arxiv.org/abs/2604.05939v1

8. FrontierFinance: Long-Horizon Financial Tasks Benchmark 首个评估LLM在真实金融任务中计算机使用能力的基准，填补AI劳动力替代研究空白。 🔗 http://arxiv.org/abs/2604.05912v1

9. BOSCH: Black-Box Binary Optimization for Attention-Head Selection 通过二进制优化方法自动选择短上下文注意力头，在保持性能前提下显著降低推理KV缓存开销。 🔗 http://arxiv.org/abs/2604.05942v1

10. TriAttention: Trigonometric KV Compression for Long Reasoning 针对长推理场景的KV缓存瓶颈，提出三角函数压缩方法，高效处理长上下文推理。 🔗 http://arxiv.org/abs/2604.04921v1
